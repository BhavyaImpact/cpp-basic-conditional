<h3>How ASCII Values Help in Programming</h3>

<pre>
How ASCII Values Help in Programming

ASCII (American Standard Code for Information Interchange) assigns a numeric value to every character.
That means characters like A, a, 0, ! are all stored as numbers inside the computer.

So when you compare characters in C++, you are actually comparing their ASCII numbers.

<h3>How ASCII Works</h3>

Every character has a unique number:

Character 	   ASCII Value
    A     -->	    65
    Z	    -->     90
    a	    -->     97
    z	    -->     122
    0   	-->     48
    9	    -->     57
  
  <p>This are the ranges that varies for (A to Z) , (a to z ) and for numbers ( 0 to 9) and the given arrow determine the ASCII values.</p>
  <h3>Example ( how to fetch)</h3>
  char ch = 'A';
  cout << (int)ch;

  <h3>Output</h3>
    --> 65
</pre>

<p>
ASCII values help determine the type of character by comparing its numeric representation.
</p>
